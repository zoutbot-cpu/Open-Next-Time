# Fn Shortcuts

Confirmed / observed:

```text
Fn + 1 = Bluetooth profile 1
Fn + 2 = Bluetooth profile 2
Fn + 3 = Bluetooth profile 3
Fn + 4 = 2.4GHz USB receiver / dongle mode
Fn + 5 = wired / main typing mode

Fn + Space = RGB control layer
Fn + Space + Left Arrow = switch RGB mode
Fn + Space + Up/Down Arrow = adjust RGB brightness

Fn + Top Delete = stop/start RGB
Fn + Insert = previous RGB profile
Fn + Page Up = next RGB profile

Fn + F1 = open media player
Fn + F2 = volume down
Fn + F3 = volume up
Fn + F4 = mute
```

Suspected / unconfirmed:

```text
Fn + F5 = possibly brightness down
Fn + F6 = possibly brightness up
```
